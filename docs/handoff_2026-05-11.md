Important packaging notes:

- `franken_networkx` is copied into the patched source tree so Nix vendoring
  does not try to vendor it as a git workspace dependency.
- Unused optional `fnx-runtime` feature edges for `ftui` and `asupersync` are
  patched out because they point to dependencies unavailable in the Nix build
  path and are not needed by the `frankenmermaid` CLI.
- The package uses the existing repo `fenixToolchain`, because upstream
  requires Rust 1.95 and `pkgs.rustPlatform` was using rustc 1.94.
- Native build support includes `cmake`, `libclang`, bindgen include args, and
  runtime wrapping for `libstdc++.so.6`.
